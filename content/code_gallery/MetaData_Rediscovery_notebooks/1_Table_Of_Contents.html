Explore a Vessel Class Prefix PP (PP-nnn)¶

Individual Ships (nnn) are available in each vessel class exploration as well as an aggregate of each vessel class. These explorations are identified prefixes from FOSDIC records. Initial explorations used R3 records to make these plots.

Some Prefixes are clearly invalid (non-numeric) while others appear valid but have minimal ships/samples to identify, and others only have a single sample or two and may be improperly encoded. All but the clearly invalid prefixes are shown below:
