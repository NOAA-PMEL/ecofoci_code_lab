EcoFOCIpy SBE16 QC example (seperated)¶

see (EcoFOCIpy_19ckp12a_sbe16_6629.ipynb) for initial processing and formatting. Data would have been convertted from raw instrument format to csv files, initial dirty plots created and a meta-data rich and meta-data poor dataset would have been created. The metadata poor csv file is used for the initial preliminary data archive, the metadata rich netcdf file is used for QC procedures and eventual final data archive.

Two Options

  • use xarray to load the netcdf data directly (working file), or pandas to load the csv file (initial archive)

  • use erddapy to load the ERDDAP hosted dataset
