EcoFOCIpy SBE16 19BS-5A 7297 example (seperated)¶

see EcoFOCIpy_sbe_UAF.ipynb for initial processing and formatting. Data would have been convertted from raw instrument format to csv files, initial dirty plots created and a meta-data rich and meta-data poor dataset would have been created. The metadata poor csv file is used for the initial preliminary data archive, the metadata rich netcdf file is used for QC procedures and eventual final data archive.
